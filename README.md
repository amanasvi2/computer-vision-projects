# Computer Vision Projects

A collection of projects from my Intro to Computer Vision coursework, covering classical computer vision, deep learning, point clouds, and neural rendering.

## Projects

1. **Hybrid Images and Convolution**  
   Implemented image filtering and hybrid image generation.

2. **SIFT and RANSAC Feature Matching**  
   Built local feature matching and robust alignment pipelines.

3. **Deep Learning Recognition**  
   Explored image recognition using deep learning methods.  
   Note: only the report is included because the full code submission was too large to export.

4. **Semantic Segmentation**  
   Implemented segmentation models and evaluated visual outputs.

5. **PointNet Point Cloud Classification**  
   Worked with 3D point cloud classification using PointNet-style architectures.

6. **Neural Radiance Fields / NeRF**  
   Implemented neural rendering concepts and generated novel view outputs.

## Notes

Large datasets, model checkpoints, submission zips, and environment files are excluded from this repository.
